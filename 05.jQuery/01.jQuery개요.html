jQuery 개요

* jQuery란?

기존의 자바스트립트 언어만으로 구현하기에는 복잡했던 구문 작성들을
간편하게 작성하기 위해 '존 레식'에 의해 만들어짐
JavaScript기반 라이브러리(유용한 기능들의 모음)
즉. JavaScript언어로 유용한 함수나 내용들이 이미 작성되어 있는 모음집

복잡한 자바스크립트 구문을 보다 짧고 쉽게 사용 가능
> jQuery의 장점
1) DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있음
2) AJAX(비동기식 통신), 이벤트 처리 등등 폭넓게 지원함
3) jQuery와 관련된 확장형 플러그인, 오픈 api등을 지원
(차트, 슬라이드, 등등...)

* jQuery 라이브러리 연결 방법

head태그내에 script태그로 외부 jQuery관련 .js파일 연결하기
단, 오프라인/온라인 방식 있음
- jQuery 공식사이트 : 공식 사이트
1. jQuery 라이브러리를 직접 다운로드 받은 후 경로지정(오프라인)

2. CDN(Content Delivery Network) 방식(온라인)
파일을 다운로드 받지 않고 라이브러리를 사이트의 url을 파일 경로에 지정하여 사용

* jQuery 구문 작성하기

jQuery 구문으로 dom요소를 다루는 구문을 기본적으로 작성하기 때문에
문서상에 요소들이 다 만들어진 후 실행되어야 함

> 순수 자바 스크립트에서는 

      window.onload = function() {
        실행내용;
      }

      javascript에서의 window.onload 구문은
      해당 문서의 요소들이 전부 로드될 뿐만 아니라
      연동되어 있는 외부문서들도 전부 가지고 온 뒤에 실행됨
      또한 window.onload는 단 한번만 실행(작성) 됨
> jQuery에서의 해당 내용은 3가지 구문으로 제공 
      1) jQuery(document).ready(function() {
          실행내용;
          });

      2) $(document).ready(function() {
          실행내용;
          });

      3) $(function() {
          실행내용;
          });

      해당 문서의 요소들만 다 로드되면 곧바로 실행됨
      여러번 사용가능(제한없음). 순서대로 실행.
      즉시실행함수와 유사함
